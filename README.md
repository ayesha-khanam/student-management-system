# Student Management System

## Description
This is a console-based Student Management System developed in Python. The program allows users to manage student records using a simple menu-driven interface. It supports adding, viewing, searching, updating, and deleting student information. The data is stored in a JSON file so that records remain saved even after the program is closed.

## Features
- Add new student records
- View all students
- Search student by ID
- Update student information
- Delete student records with confirmation
- Automatic student ID generation
- Data stored using JSON file
